# ネタレコ
## サイト概要
### サイトテーマ
芸人のおすすめネタを共有できるレビューサイト
​
### テーマを選んだ理由
好きな芸人さんができ、配信サイトでその芸人さんのネタをよく見ていたのですが、自分の好きな芸人さん以外のネタを見る機会がなく、新規開拓があまりできていない状況でした。そんな中、友人のおすすめで、今まで見たことが無かった芸人さんのネタを見ました。そのネタがとにかく面白く、人におすすめしたいという友人の気持ちに共感しました。  
芸人さんがSNSなどを利用してネタを発信することも多い昨今、ネタを見る機会が増えています。そのため、芸人さんの存在やネタを知り、ファンを増やすことで劇場に通う人も増えるのではと考えました。面白いネタを共有することで、新規ファン獲得やファン同士での繋がりを広げることができるサイトができればと思い、このテーマにしました。
​
### ターゲットユーザ
・賞レースなどから芸人を知り、どのネタが面白いのか知りたい人  
・自分の好きな芸人のネタを共有したい人  
・とにかく笑いたい人
​
### 主な利用シーン
・お腹を抱えて笑いたいとき  
・自分の好きなネタを布教したいとき
